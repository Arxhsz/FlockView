FlockView Web Flasher

Flash the official FlockViewScanner v1.0.0 firmware to an ESP32-WROOM-32 development board from a Chromium-based browser using Web Serial.

Install Firmware

Connect the ESP32 over USB, hold BOOT if your board requires it, then choose the serial port and install the firmware.

1 Use Chrome or Edge

Web Serial is required. Safari and Firefox do not expose the needed serial flashing API.

2 Plug in the ESP32

Use a data-capable USB cable and close PlatformIO Serial Monitor or any other app using the port.

3 Connect in FlockView

After flashing, open the Mac app and use Hardware Mode to connect at 115200 baud.

Firmware: FlockViewScanner v1.0.0 Board: ESP32-WROOM-32 / esp32dev Baud: 115200 Mode: passive Wi-Fi + BLE scanner
